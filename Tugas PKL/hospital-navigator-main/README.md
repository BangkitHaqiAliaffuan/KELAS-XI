# Hospital Navigator

A modern application to navigate hospital floors and departments easily.

## Features

- Interactive Floor Selection
- Department Search and Categorization
- Detailed Location Information
- Mobile-Responsive Design

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
